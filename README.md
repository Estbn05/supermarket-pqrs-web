# Frontend PQRS

Aplicacion Angular/PWA para clientes y gestores.

## Ejecutar

```powershell
npm install
npm start
```

La app queda en `http://localhost:4200`.

## Rutas

- `/cliente/radicar`: formulario de radicacion PQRS.
- `/cliente/login`: login del cliente.
- `/cliente/historial`: historico del cliente.
- `/gestor/login`: login del Gestor PQRS.
- `/gestor/panel`: listado, filtros, cambio de estado, anexos y reporte PDF.
